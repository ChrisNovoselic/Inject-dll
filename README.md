# Inject-dll
C++ 11 std inject dll into other dll

Скрывает описание процесса из приложения Диспетчер задач (вкладка - Процессы).
Диспетчер задач должен выполняться.
При останове/запуске Диспетчера задач, внедренная ранее библиотека отсоединяется, все процессы отображаются в обычном порядке.

Сборка MSVC  2015 Community (только x64).

Обязательны 2 аргумента командной строки
1) наименование встраиваемой в целевой объект библиотеки
2) зарезервировано - наименование образа процесса, который следует скрыть

Проекты и файлы в решении:
1) hookx86 - встравивемая библиотека для ОС Windowsx86
2) hookx64 - x64
3) injectdll - внедряет указанную в 1-ом аргументе библиотеку в процесс 'taskmgr.exe'
ВАЖНО! Библиотека должна находиться по размещению исполняемого модуля
